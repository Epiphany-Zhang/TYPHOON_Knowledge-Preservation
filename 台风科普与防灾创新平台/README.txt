📂 目录结构：
├── dist/            ⭐ 这是生产构建（推荐直接使用）
│   ├── index.html   → 在浏览器中打开此文件以预览网站
│   ├── assets/      → 打包的 CSS、JS 和其他资源文件
│   └── ...
├── src/             📝 源代码目录（用于开发和修改）
│   ├── components/  → React 组件
│   ├── App.jsx      → 主应用组件
│   ├── index.js     → 入口文件
│   └── ...
├── package.json     📦 项目依赖配置
├── README.md        📖 项目文档
└── ...

🚀 快速开始（两种选择）：
选项 1：直接使用（推荐给初学者）
1. 进入 dist/ 目录
2. 双击 index.html 文件
3. 网站将在您的浏览器中打开！
