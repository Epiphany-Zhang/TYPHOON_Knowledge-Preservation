📂 目录结构：
├── dist/            ⭐ 这是生产构建（推荐直接使用）
│   ├── index.html   → 在浏览器中打开此文件以预览网站
│   ├── assets/      → 打包的 CSS、JS 和其他资源文件
│   └── ...
├── src/             📝 源代码目录（用于开发和修改）
│   ├── components/  → React 组件
│   ├── App.jsx      → 主应用组件
│   ├── index.js     → 入口文件
│   └── ...
├── package.json     📦 项目依赖配置
├── README.md        📖 项目文档
└── ...

🚀 快速开始（两种选择）：
选项 1：直接使用（推荐给初学者）
1. 进入 dist/ 目录
2. 双击 index.html 文件
3. 网站将在您的浏览器中打开！

选项 2：开发模式（适合有编码经验的用户）
1. 确保已安装 Node.js（推荐版本：16+）
2. 在项目根目录打开终端/命令行
3. 运行：npm install
4. 运行：npm start
5. 开发服务器将自动在您的浏览器中打开

💡 提示：
- dist/ 目录包含优化后的构建，可以直接部署到服务器
- 根目录下的 index.html 不是最终版本 — 始终使用 dist/ 内的文件
- 要修改代码，请编辑 src/ 内的文件并重建项目
- 项目使用 TypeScript 和 Tailwind CSS 技术栈
